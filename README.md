# Matrix Computations for Large-Scale Systems (SF2524)

This repository contains coursework, computational implementations, and analytical reports developed for the Matrix Computations for Large-Scale Systems course as part of the MSc in Applied and Computational Mathematics at KTH Royal Institute of Technology.

The implementations focus on numerical linear algebra, exploring the efficiency, stability, and convergence of iterative methods for massive matrices where direct solvers are computationally prohibitive.



---

## Repository Structure

### 📁 [Homework 1: Eigenvalue Algorithms & Krylov Subspace Methods](./Homework_1_Eigenvalue_Algorithms)
Explores eigenvalue extraction techniques, the power method, and the numerical stability of various Gram-Schmidt orthogonalization procedures. Features implementations of the Arnoldi iteration and Shift-and-Invert methods.

### 📁 [Homework 2: Iterative Solvers for Linear Systems](./Homework_2_Iterative_Linear_Solvers)
Focuses on Krylov subspace methods for solving large, sparse linear systems and least squares problems. Includes custom implementations of GMRES, Conjugate Gradient (CG), and CG for Normal Equations (CGN).

### 📁 [Homework 3: The QR Algorithm & Matrix Functions](./Homework_3_QR_and_Matrix_Functions)
Explores advanced eigenvalue solvers and algorithms for evaluating functions of matrices. Features the shifted QR algorithm, Hessenberg reductions, the Schur-Parlett method, and Taylor series approximations for the matrix exponential.
